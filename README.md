# anitadayoff
Determine days off 

### Install
```
yarn install
```
1. create a `input.csv` file to read dates with the headers:
```
year,month,day
2020,12,25
```

### Run
```
yarn start
```

Or optionally your own csv file
```
yarn start INPUT_FILE="another.csv"
```
